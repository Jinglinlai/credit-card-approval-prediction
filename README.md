# Credit Card Approval Prediction

This project uses Logistic Regression and GridSearchCV to predict credit card approval decisions.

## Workflow
- Data preprocessing (missing values handled using mean/mode)
- One-hot encoding for categorical features
- Feature scaling using StandardScaler
- Model training using Logistic Regression
- Hyperparameter tuning using GridSearchCV

## Model Performance
- Best CV Score: 0.8247078073866293
- Final Test Accuracy: 0.8026315789473685

## Tech Stack
- Python
- Pandas
- Scikit-learn
